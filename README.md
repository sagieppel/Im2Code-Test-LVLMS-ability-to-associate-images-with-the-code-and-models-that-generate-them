# -Im2Code-Test-LVLMS-ability-to-associate-images-with-the-code-and-models-that-generate-them
This is done using  a multi choice question where the LVLM (GPT,Qwen,Gemini,Grok) receives code or model verbal description and several images generated by the model, as well as images generateby by different models, and is asked to find which images were generated by the described model or code.
